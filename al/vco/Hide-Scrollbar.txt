Utiliser "-webkit-scrollbar" pour autoriser à scroller au lieu de "overflow:hidden"

Appliquer à l'ensemble de la page :
gigaGalerieClient/css/sidebar.css :

Ajouter :

::-webkit-scrollbar { 
    display: none; 
}

OU 

/* CHROME */
#sidebar-wrapper::-webkit-scrollbar { 
    display: none; 
}

/* FIREFOX */
#sidebar-wrapper {
    overflow : -moz-scrollbars-none
}

Pour l'appliquer uniquement à la sidebar
